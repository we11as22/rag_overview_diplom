## Текущий фокус промежуточной сдачи

На текущем этапе реализован и измерен самый распространённый прикладной сценарий RAG: **бот по документации**. Он закрывает базовую часть диплома: загрузка корпуса, чанкинг, sparse/dense/hybrid retrieval, подбор параметров, прототип агента и воспроизводимый экспериментальный контур.

Следующий этап диплома — расширить обзор за пределы классического documentation QA: добавить сравнение более продвинутых техник RAG и описать, где они применимы.
